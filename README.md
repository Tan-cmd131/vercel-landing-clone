# 🚀 Vercel Landing Page Clone

A responsive landing page inspired by the Vercel website, built using **HTML** and **CSS** to practice modern frontend layout, styling, and deployment workflows.

---

## 🌐 Live Demo

🔗 https://vercel-landing-clone-eta.vercel.app

---

## 📌 Features

* Dark themed modern UI
* Responsive landing page layout
* Hero section with call-to-action buttons
* Feature cards section
* Multi-column footer
* Clean navigation bar
* Fully deployed using Vercel

---

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox & Grid Layout
* Git & GitHub
* Vercel Deployment
